Wang, L., Yu, H. B., Hu, J., Theeuwes, J., Gong, X., Xiang, Y., … Zhou, X. (2015). Reward Breaks Through Center-Surround Inhibition via Anterior Insula. Human Brain Mapping, 36(12), 5233–5251. https://doi.org/10.1002/hbm.23004
